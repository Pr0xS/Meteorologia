\hline

La función \hyperlink{meteo_8h_a3d3bf46df3c9bf070f7fd007f03e0308}{load\+Data()} devuelve el número de elementos cargados si tiene éxito. En caso de error devolverá -\/1. 
\end{DoxyReturn}
\begin{DoxyWarning}{Atención}

Para ello liberaremos la información para cada entrada meteorológica para cada ciudad y posteriormente liberaremos la memoria reservada para cada ciudad.
